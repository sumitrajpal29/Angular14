## Creating new Angular *Workspace*
`ng new name`

To open all in VScode 
`cd name`
`code .`

## Adding *.gitignore* after pushing into github by mistake
`git rm -r -cached .`
`Add .gitignore file or required updates in .gitignore`
`add ., commit and push`
